# 🔹 1. DATA UNDERSTANDING

## What is a Dataset?

A dataset is a collection of structured data (rows and columns).

* Rows → Observations (customers)
* Columns → Features (age, income, etc.)

---

## Types of Data

### 1. Numerical

* Continuous → total_spent, age
* Discrete → total_purchases

### 2. Categorical

* Nominal → gender, region
* Ordinal → loyalty_tier (Bronze, Silver, Gold)

---

# 🔹 2. DATA CLEANING

## Why Cleaning?

Raw data contains:

* Missing values
* Incorrect values
* Noise

---

## Handling Missing Values

### Numerical:

```python
df['column'].fillna(df['column'].mean(), inplace=True)
```

### Categorical:

```python
df['column'].fillna(df['column'].mode()[0], inplace=True)
```

---

## Why Mean?

* Keeps distribution stable
* Simple and effective

---

# 🔹 4. OUTLIER DETECTION

## What is an Outlier?

Extreme values far from normal data

---

## Boxplot

```python
sns.boxplot(x='responded', y='total_spent', data=df)
```

👉 Circles = Outliers

---

## Why Outliers Matter?

* Can distort model
* But sometimes important (high-value customers)

---

## Handling Outliers

### Option 1: Keep

👉 If meaningful (like high spenders)

### Option 2: Transform

```python
df['log_total_spent'] = np.log1p(df['total_spent'])
```

👉 Reduces skewness

# 🔹 6. HYPOTHESIS TESTING

## What is Hypothesis?

### Null Hypothesis (H₀)

No relationship

### Alternative Hypothesis (H₁)

Relationship exists

---

## P-value Concept

| P-value | Meaning         |
| ------- | --------------- |
| < 0.05  | Significant     |
| > 0.05  | Not significant |

---

## 6.1 T-Test (Numerical vs Target)

Used for:

* Comparing means between 2 groups

```python
ttest_ind(group1, group2)
```

---

## 6.2 Chi-Square Test (Categorical vs Target)

```python
chi2_contingency(table)
```

---

## Interpretation

* Reject H₀ → relationship exists
* Fail to reject H₀ → no strong evidence
